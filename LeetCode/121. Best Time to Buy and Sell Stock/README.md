Solution:  

这道题的核心方法是，我们遍历整个数组，用当前正在访问的价格，减去截止到当前为止，前面的所有天数中的最低价格。之所以这样，是因为，我们要卖股票的时候，肯定手里已经买了一个了。“已经买了一个”这个事件，肯定发生在当前这天之前。所以在程序里，我们要用一个变量保存 直到当前遍历元素为止时的最小值，然后用当前遍历的元素，也就是当天的价格，减去前面所有天数中的最小值，作为截止到当前遍历元素为止的最大收益。
